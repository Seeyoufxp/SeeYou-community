# 之友(SeeYou)开发者社区

## 功能简述

1. 首页
   - 欢迎(第n天,天气(function calling每日获取))
   - AI开发助手(RAG)
   - 全站搜索(ES)
2.社交圈子
   - 帖子
   - 聊天室
3. 知识中心
   - 技术博客
   - 问答广场
4. 个人
   - 信息
5. 后续计划
   - 积分/(经验)等级系统
   - 加入你们

## 微服务模块

1. 网关微服务 (seeyou-gateway)
2. 用户微服务 (seeyou-user)
3. 内容微服务 (seeyou-content)
4. 聊天微服务 (seeyou-chat)
5. 搜索微服务 (seeyou-search)
6. AI微服务 (seeyou-ai)

## 技术栈

- RocketMQ
  - 发帖等操作后通过消息队列通知搜索服务写入ES
  - 聊天室使用消息队列广播消息
- SpringAI
  - 每日用户首次访问欢迎语使用Function Calling调取注册时长,天气信息(欢迎语存入redis,当天有效)
- XXL-JOB
  - 获取每日天气
- 多线程/锁
  - 用户签到/发帖后,多线程(池)异步加积分等
  - 统计聊天室人数等功能需要需要用ConcurrentHashMap/AtomicLong等线程安全容器


## 表结构

- seeyou_user
  - user_info
        ```sql
        CREATE TABLE `seeyou_user_info` (
            `id` bigint(20) NOT NULL AUTO_INCREMENT COMMENT '用户ID',
            `username` varchar(50) NOT NULL COMMENT '用户名',
            `password` varchar(255) NOT NULL COMMENT '密码',
            `email` varchar(100) DEFAULT NULL COMMENT '邮箱',
            `phone` varchar(20) DEFAULT NULL COMMENT '手机号',
            `avatar` varchar(255) DEFAULT NULL COMMENT '头像URL',
            `bio` varchar(255) DEFAULT NULL COMMENT '个人简介',
            `created_at` timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
            `updated_at` timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
            PRIMARY KEY (`id`),
            UNIQUE KEY `username` (`username`),
            UNIQUE KEY `email` (`email`),
            UNIQUE KEY `phone` (`phone`)
        ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户表';
        ```
