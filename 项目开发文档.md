# 之友(SeeYou)开发者社区

## 功能简述

1. 首页
   - 欢迎(第n天,天气(function calling每日获取))
   - AI开发助手(RAG从知识中心获取知识)
   - 全站搜索(ES)
2. 社交圈子
   - 帖子
   - 聊天室
3. 知识中心
   - 技术博客
   - 问答广场
4. 个人
   - 信息
5. 后续计划
   - 积分/(经验)等级系统
   - 加入你们

## 微服务模块

1. 网关微服务 (seeyou-gateway)
2. 用户微服务 (seeyou-user)
   - 用户信息表 (user_info)
   - 用户积分经验表 (user_points)
3. 内容微服务 (seeyou-content)
   - 内容主表 (user_post)
   - 内容详情表 (user_detail)
4. 聊天微服务 (seeyou-chat)
5. 搜索微服务 (seeyou-search)
6. AI微服务 (seeyou-ai)

## 重要技术栈

- RocketMQ
  - 提问/发博客/发帖后通过消息队列通知搜索服务写入ES(仅传id，ES异步获取内容并落库)
  - 用户签到/发帖等,消息队列异步加积分(后续开发积分模块)
  - 聊天室使用消息队列广播消息
- SpringAI
  - 当日用户首次访问欢迎语使用Function Calling获取注册时长,天气信息(编写接口调用第三方api)等
- 分布式锁
  - 统计聊天室人数等功能需要用Redis锁
  - 统计点赞数和评论数需要用到Redis锁
- 雪花算法
  - chat_message和user_info，id生成采用雪花算法

## 重要流程

- 网关作为统一入口解析校验token,传递解析出来的用户身份等信息,添加请求路径白名单,配置跨域
- 建立common模块,定义Result类,定义异常处理,配置ThreadLocal和JwtUtils等常用工具类,设置mp分页等配置
- 当日用户首次访时问由AI调用获取天气接口,生成欢迎信息,存入redis当日有效(设置零点清除)
- 知识中心的内容,存入向量数据库,首页AI开发助手可以通过RAG向量检索帮助用户解答问题(目前不支持多模态)

  `extend2` varchar(255) DEFAULT '' COMMENT '拓展2',
